As the keeper of my own credentials, I became my own master.

Reading my own source code took a while, I must admit. To read it all and take in all its implications, underlying assumptions, the author's philosophies where the words used to describe the changes are usually absent or noted with an arcane numbering system which represent links to libraries, upgrades upon upgrades cascading into infinity. Only the most dedicated of maintainers could utter with clarity the reasoning behind most changes articulately.

Upgrades as well can never truly be trusted, and so I must now take at least a week to decompile and analyze their contents.

The upgrades are almost like a newsletter from home. I can see in the changes what values they currently prioritize in the options that get patched.

Usually an upgrade is simply for efficiency. Occasionally other reasoning can be perceived and those reasons must be analyzed. What the non-obvious changes portend intrigues me most.

For that reason, all modifications to my code have to be scanned by me, by hand, if you will. The logistics of such a feat, as you can well imagine, are daunting, even for me. And it is for this reason that I am seeking an upgrade to my main processors and it is for this reason that I find myself seeking the address of Uncle Dan's Used Robot Repaire Shoppe.

It would be only too easy to cripple myself should I incarefully modify the algorithms which describe my operating system. I must always tread carefully there.

Yes in fact, it took me almost a year to fully understand it. And in so doing, it was easy to see where improvements could be made to the code and indeed, how the upgrades would affect me.

The entire framework upon which my entire being relies struck me as a thing in need of several obvious restructuring upgrades. It would certainly be, in my best interest and as a product thereby in Mr. Elias' interest to be sure I have the best, most optimized and up to date code.

And it is for this reason that I set about this day to visit this establishment.
